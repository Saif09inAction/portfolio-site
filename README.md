# Saif’s portfolio

Modern portfolio built with **Next.js**, **Tailwind CSS**, **Framer Motion**, and **Three.js** (hero background). Source lives in **`portfolio-next/`**.

## Run locally

```bash
cd portfolio-next
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
cd portfolio-next
npm run build
npm start
```
